# Number of rooted / unrooted tree shapes

`nRootedShapes` and `nUnrootedShapes` give the number of (un)rooted
binary trees on *n* unlabelled leaves.

## Usage

``` r
nRootedShapes

nUnrootedShapes
```

## Format

An object of class `integer64` of length 55.

An object of class `integer64` of length 60.

## Source

`nRootedShapes` corresponds to the Wedderburn-Etherington numbers, [OEIS
A001190](https://oeis.org/A001190)

`nUnrootedShapes` is [OEIS A000672](https://oeis.org/A000672)
