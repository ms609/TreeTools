# Double factorials

A vector with pre-calculated values of double factorials up to 300!!,
and the logarithms of double factorials up to 50 000!!.

## Usage

``` r
doubleFactorials
```

## Format

An object of class `numeric` of length 300.

## Details

301!! is too large to store as an integer; use `logDoubleFactorials`
instead.
