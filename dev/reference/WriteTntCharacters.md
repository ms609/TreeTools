# Write morphological character matrix to TNT file

Write morphological character matrix to TNT file

## Usage

``` r
WriteTntCharacters(
  dataset,
  filepath = NULL,
  comment = "Dataset written by `TreeTools::WriteTntCharacters()`",
  types = NULL,
  pre = "",
  post = ""
)

# S3 method for class 'phyDat'
WriteTntCharacters(
  dataset,
  filepath = NULL,
  comment = "Dataset written by `TreeTools::WriteTntCharacters()`",
  types = NULL,
  pre = "",
  post = ""
)

# S3 method for class 'matrix'
WriteTntCharacters(
  dataset,
  filepath = NULL,
  comment = "Dataset written by `TreeTools::WriteTntCharacters()`",
  types = NULL,
  pre = "",
  post = ""
)
```

## Arguments

- dataset:

  Morphological dataset of class `phyDat` or `matrix`.

- filepath:

  Path to file; if `NULL`, returns a character vector.

- comment:

  Optional comment with which to entitle matrix.

- types:

  Optional list specifying where different data types begin.
  `c(num = 1, dna = 10)` sets characters 1..9 as numeric, 10..end as
  DNA.

- pre, post:

  Character vector listing text to print before and after the character
  matrix. Specify `pre = 'piwe=;` if the matrix is to be analysed using
  extended implied weighting (`xpiwe=`).
