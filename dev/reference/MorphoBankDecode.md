# Decode MorphoBank text

Converts strings from MorphoBank notes into a Latex-compatible format.

## Usage

``` r
MorphoBankDecode(string)
```

## Arguments

- string:

  String to process

## Value

`MorphoBankDecode()` returns a string with new lines and punctuation
reformatted.
